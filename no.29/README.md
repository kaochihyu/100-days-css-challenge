#### Point
* import fontawesome 的 搜尋 icon
* 改變 placeholder 的樣式要用選擇器:: placeholder
* search input 的框線用 :focus-visible { outline: none; } 來去除掉 
* 用 @import 把 google font、font awesome 在 css 檔案裡面引入

* SCSS
