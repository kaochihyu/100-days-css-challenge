#### Point
- svg的 viewBox="x y width height"，x y 是視角畫線的起始位置
- 方框線用 polyline 來畫
- dash-array: n1 n2， n1 是線的長度，n2 是間距
- 用 dash-strokeoffset 的變化加上 transition 來讓線條產生變化
